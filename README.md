## 🚀 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📦 部署

本网站部署在 Vercel：

```bash
# 安装 Vercel CLI
npm i -g vercel

# 部署
vercel
```
