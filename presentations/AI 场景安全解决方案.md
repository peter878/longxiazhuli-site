# AI 场景安全解决方案

## 让企业放心使用 AI，安心建设大模型

---

# 第 1 页：AI 应用的两大安全场景

## 场景一：使用 AI 的安全（90% 企业的需求）

**典型场景**：

| 场景 | 安全风险 | 我们的防护 |
|------|----------|------------|
| **AI 客服** | 输出错误承诺、泄露内部信息 | 输出内容审核 + 敏感词过滤 |
| **AI 办公** | 员工上传机密文档、AI 泄露数据 | 输入文档脱敏 + 输出审查 |
| **AI 营销** | 生成违规广告、虚假宣传 | 文案合规审核 + 事实核查 |
| **AI 编程** | 生成有漏洞的代码、泄露 API Key | 代码安全扫描 + 密钥检测 |

**核心能力**：
- ✅ **输入审核**：检测用户上传的文件、提问内容是否包含敏感信息
- ✅ **输出审核**：AI 回答前进行安全检查，防止不当内容流出
- ✅ **安全代答**：遇到敏感问题时，不直接拒答，而是正向引导

---

# 场景二：自建大模型的安全（大型企业需求）

**典型场景**：

| 场景 | 安全风险 | 我们的防护 |
|------|----------|------------|
| **训练平台** | 训练数据投毒、模型被窃取 | 数据清洗 + 模型加密 + 访问控制 |
| **推理平台** | 提示词注入、越狱攻击、算力滥用 | 请求审核 + 速率限制 + 异常检测 |
| **算力集群** | 未授权访问、资源盗用 | 身份认证 + 网络隔离 + 审计日志 |

**核心能力**：
- ✅ **平台安全**：训练/推理平台的访问控制、网络隔离
- ✅ **模型安全**：模型加密、防窃取、防逆向
- ✅ **数据安全**：训练数据脱敏、推理数据不留存
